# well-pixi

A well-configured PixiJS game template for jams and small projects.

Nothing groundbreaking - just a reliable, nicely set up starting point that gets out of your way.

Based on [create-pixi](https://github.com/pixijs/create-pixi).

## Origin

Born from [Depth of Moments](https://ldjam.com/events/ludum-dare/57/depth-of-moments) ([source](https://github.com/nnmbzr/ludum-dare-57)), a Ludum Dare 57 entry - the approaches and setup from that jam grew into this template.
